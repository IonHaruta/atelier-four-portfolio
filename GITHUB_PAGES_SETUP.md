# Configurare GitHub Pages — O singură setare

## Pasul unic (obligatoriu)

1. Mergi la: **https://github.com/IonHaruta/atelier-four-portfolio/settings/pages**
2. La **Build and deployment** → **Source** alege **GitHub Actions** (nu "Deploy from a branch")
3. Gata. Nu mai trebuie nimic altceva.

---

După ce faci asta, la fiecare push pe `main` workflow-ul va construi și deploya site-ul automat.

**Default branch** trebuie să rămână **main** (Settings → General).
